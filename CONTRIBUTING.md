# Contribution Guide

## Dev

```bash
yarn install

# watch and build TypeScript into ./build
yarn dev

# open a browser with the extension loaded. It will automatically be reloaded when the files inside ./build changed
yarn dev:firefox
yarn dev:chrome
```
